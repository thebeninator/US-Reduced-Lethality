# US Reduced Lethality
Gives M1/IPs and M60A1/3s weaker ammunition. Features M774 APFSDS-T, M735 APFSDS-T, M392A2 APDS-T, and M393A2 HEP-T. Which AP rounds are used can be customized. 

# Default loadouts

M1/IP:
- M774 APFSDS-T
- M456 HEAT-FS-T

M60A1/3: 
- M392A2 APDS-T
- M456 HEAT-FS-T
- M393A2 HEP-T
